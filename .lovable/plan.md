

# تحسين أداء زر "إضافة مواد الشبكات" وإضافة الاقتراحات

## المشاكل الحالية

1. **لا توجد حماية من التكرار**: الضغط على الزر عدة مرات يضيف نفس البيانات مكررة
2. **لا يوجد مؤشر تقدم**: إضافة ~80 مادة تتم على دفعات لكن المستخدم لا يرى التقدم
3. **لا يتم تعطيل الزر أثناء التنفيذ**: يمكن الضغط مرة أخرى قبل انتهاء العملية
4. **لا يوجد فحص للبيانات الموجودة**: يجب التحقق إن كانت المواد موجودة بالفعل قبل الإضافة

## الحلول المقترحة

### 1. فحص التكرار قبل الإضافة (`useSampleLibraryData.tsx`)

- قبل إدراج البيانات، فحص إذا كانت مواد بنفس الـ `source` موجودة مسبقاً
- إذا وُجدت: إظهار رسالة تأكيد "البيانات موجودة بالفعل، هل تريد إضافتها مرة أخرى؟"
- إضافة دالة `checkExistingData` تستعلم عن عدد السجلات بنفس المصدر

### 2. مؤشر تقدم أثناء الإضافة (`LibraryDatabase.tsx`)

- إضافة state لنسبة التقدم `progress`
- عرض شريط تقدم (Progress bar) في حوار التأكيد أثناء الإدراج
- تحديث النسبة بعد كل دفعة (batch) من الـ 30 عنصر

### 3. تحسين تجربة الأزرار

- تعطيل جميع أزرار الإضافة أثناء تنفيذ أي عملية إضافة
- إظهار عدد العناصر التي ستُضاف بوضوح على الزر
- إضافة أيقونة تحميل متحركة أفضل مع نص يتغير ("جاري الإضافة...")

### 4. إضافة زر "إضافة الكل" مجمع

- زر واحد يضيف مواد الشبكات + عمالة ومعدات الشبكات معاً
- يعرض ملخص شامل لما سيُضاف

## التفاصيل التقنية

### الملفات المتأثرة

| الملف | التغيير |
|-------|---------|
| `src/hooks/useSampleLibraryData.tsx` | إضافة فحص التكرار + callback للتقدم |
| `src/components/LibraryDatabase.tsx` | تحسين الأزرار + شريط تقدم + زر إضافة الكل |

### تغييرات `useSampleLibraryData.tsx`

- إضافة دالة `checkExistingNetworkMaterials()` تستعلم عن `material_prices` بـ `source = 'water_sewage_data'`
- تعديل `addWaterSewageMaterials` لتقبل callback للتقدم: `onProgress?: (percent: number) => void`
- استدعاء `onProgress` بعد كل batch
- إضافة دالة `addAllNetworkData()` تجمع المواد + العمالة + المعدات

### تغييرات `LibraryDatabase.tsx`

- إضافة `progress` state مع `Progress` component
- تعطيل الأزرار الثلاثة أثناء أي عملية إضافة (حالة `isAnyLoading`)
- إضافة زر "إضافة جميع بيانات الشبكات" يجمع المواد والعمالة والمعدات
- عرض رسالة عند وجود بيانات مكررة مسبقاً
