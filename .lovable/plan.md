

# تحسين شاشة عروض الأسعار + إضافة اقتراحات عامة

## 1. التحقق من حالة إصلاح التحليل

الإصلاحات التالية مطبقة بالفعل وتعمل بشكل صحيح:
- تم إزالة النص البديل (fallback) عند فشل استخراج النص - يتم توجيه المستخدم لـ OCR مباشرة (سطر 657-668)
- التحقق من عدد البنود قبل تعيين الحالة: إذا كان 0 بند يبقى `pending` ويظهر رسالة OCR (سطر 707-708، 736-743)
- رسالة واضحة مع زر "إعادة التحليل بـ OCR" عند 0 بنود (سطر 1729-1745)
- تحميل `pdfjsLib` في بداية `performOCR` (سطر 458-461)

**لا يحتاج تعديل إضافي** - الإصلاحات مكتملة.

## 2. تحسين لوحة الإحصائيات

### الملف: `src/components/QuotationUpload.tsx`

لوحة الإحصائيات الحالية (سطر 1403-1423) تعرض 4 بطاقات فقط. سيتم توسيعها لتشمل:

- **عدد الموردين** (القيمة `quotationStats.suppliers` محسوبة بالفعل لكن غير معروضة)
- **متوسط سعر البند** (يحتاج حساب جديد في `useMemo`)
- تحسين التصميم: أيقونات ملونة لكل بطاقة + تدرج لوني خفيف

### التعديلات:
1. **سطر 162-168**: إضافة `avgItemPrice` إلى `quotationStats`
2. **سطر 1403-1423**: توسيع الشبكة إلى 6 بطاقات بتصميم محسن مع أيقونات

## 3. تحسين التصميم العام

### الملف: `src/components/QuotationUpload.tsx`

- تحسين منطقة الرفع: إضافة تدرج لوني وظل خفيف
- تحسين بطاقات الإحصائيات: أيقونات ملونة + حدود مميزة + تأثير hover
- تحسين رأس قائمة العروض: فصل أوضح بين العنوان وأزرار الإجراءات

## 4. إضافة اقتراحات عامة للبرنامج

### الملف: `src/components/QuotationUpload.tsx`

إضافة قسم "اقتراحات سريعة" أسفل منطقة الرفع عندما لا توجد عروض أسعار، يحتوي على:
- روابط سريعة لأقسام البرنامج المرتبطة (المكتبة، المقارنة، التقارير)
- نصائح استخدام قصيرة
- إجراءات مقترحة (رفع عرض، استيراد من مشروع، تنزيل قالب)
