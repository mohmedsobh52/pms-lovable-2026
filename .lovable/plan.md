

# تحديث شاشة المكتبة — إضافة مواد v11 واقتراحات ذكية

## المشكلة
المكتبة تفتقر إلى فئات ومواد أساسية تم تعريفها في محركات v11 (أعمال ترابية، أسفلت، طرق، إنشائي موسع)، ولا تحتوي على نظام اقتراحات ذكية.

---

## 1. إضافة فئات مواد جديدة (`useMaterialPrices.tsx`)

إضافة 6 فئات جديدة إلى `MATERIAL_CATEGORIES`:
- `earthworks` — أعمال ترابية (حفر/ردم/فرشة)
- `asphalt` — أسفلت ورصف
- `road_base` — طبقات أساس الطرق
- `road_accessories` — ملحقات الطرق (أرصفة/حواجز/إنارة)
- `concrete_works` — أعمال خرسانية (قوالب/صب)
- `safety_temporary` — أعمال مؤقتة وسلامة

إضافة مجموعتين إلى `CATEGORY_GROUPS`:
- `earthworks_asphalt` — "حفر وأسفلت"
- `road` — "طرق"

---

## 2. إضافة بيانات مواد جديدة (~60 مادة) (`useSampleLibraryData.tsx`)

إضافة مصفوفة `EARTHWORKS_ASPHALT_MATERIALS` تشمل:

**أعمال ترابية (~15 مادة):**
- حفر خنادق آلي (≤2م / 2-4م / 4-6م)
- حفر صخر ميكانيكي
- ردم رمل ناعم / ردم تربة مدموكة / ردم حجر مجروش
- فرشة رملية 150مم / حماية خرسانية
- دعم جوانب خندق / ضخ مياه جوفية
- تخلص من حفريات / نقل خارج الموقع

**أسفلت وطبقات (~15 مادة):**
- Wearing Course 50mm / 60mm
- Binder Course 60mm / 75mm
- Base Course 75mm / 100mm
- Prime Coat / Tack Coat
- أساس مجروش 200mm / 300mm
- فرشة رمل 200mm / قطع أسفلت / إزالة أسفلت / ترقيع

**ملحقات طرق (~10 مواد):**
- أرصفة مشاة 150mm
- حافة طريق kerb 150×300 / 200×400
- نيوجرسي Jersey barrier
- علامات أرضية / لوحات مرورية / أعمدة إنارة

**إنشائي موسع (~10 مواد):**
- خرسانة C20/C25/C35
- قوالب خشبية / معدنية
- شبكات حديد ملحومة

**سلامة ومؤقتات (~5 مواد):**
- شريط تحذيري / لبادة حماية
- سياج مؤقت / إضاءة مؤقتة

إضافة دالة `addEarthworksAsphaltMaterials` + زر إضافة في الـ UI.

---

## 3. إضافة اقتراحات ذكية للمكتبة (`LibraryDatabase.tsx`)

إضافة بطاقة اقتراحات ديناميكية أعلى الـ Tabs تعرض:

| الشرط | الاقتراح | الإجراء |
|---|---|---|
| المكتبة فارغة من مواد الحفر والأسفلت | "أضف مواد الأعمال الترابية والأسفلت" | زر إضافة مباشر |
| يوجد مواد شبكات بدون عمالة | "أضف عمالة ومعدات الشبكات" | فتح dialog الإضافة |
| أسعار منتهية الصلاحية > 30% | "حدّث الأسعار المنتهية" | تفعيل فلتر الصلاحية |
| لا يوجد موردين مرتبطين | "أضف موردين لتحسين المقارنات" | الانتقال لصفحة الموردين |
| يوجد مواد بدون اسم عربي | "أكمل الترجمة العربية للمواد" | تفعيل فلتر مخصص |
| المكتبة جاهزة بالكامل | "صدّر المكتبة كملف Excel احتياطي" | تنفيذ التصدير |
