

# تطوير شاشة أدوات التحليل والتسعير — طرق تسعير حديثة + إعادة ترتيب + ألوان العلامة التجارية

## الوضع الحالي

شاشة `AnalysisToolsPage.tsx` تحتوي على 4 تبويبات فقط: تحليل التكاليف، مقارنة الأسعار، مقارنة BOQ، أسعار السوق. التصميم بسيط بدون بطاقات إحصائية أو اقتراحات ذكية.

---

## التعديلات المطلوبة

### 1. إضافة طرق تسعير حديثة للبنية التحتية المتكاملة (3 أدوات جديدة)

| الأداة | الوصف | المصدر |
|---|---|---|
| **تسعير الحفر بالعمق** | حساب تكلفة الحفر حسب العمق (≤2م، 2-4م، 4-6م) مع عوامل التربة | `reference-prices.ts` + بيانات محلية |
| **تسعير الطرق المتكامل** | حساب تكلفة طبقات الطريق (Sub-base → Base → Binder → Wearing) بالكثافة والسمك | أسعار مرجعية + AI |
| **تسعير شبكات المياه/الصرف** | حساب تكلفة الشبكة شاملة الأنابيب + غرف التفتيش + القطع + إعادة الأسفلت | أسعار مرجعية |

كل أداة ستكون عبارة عن بطاقة تفاعلية بها حقول إدخال (أطوال/أعماق/أقطار) وتحسب التكلفة فوراً مع تصدير النتائج.

### 2. إعادة ترتيب الشاشة

```text
┌─────────────────────────────────────────────────┐
│  Header + Stats Cards (4 بطاقات KPI)            │
│  [إجمالي البنود] [مسعّر] [غير مسعّر] [متوسط]    │
├─────────────────────────────────────────────────┤
│  SmartSuggestionsBanner                         │
├─────────────────────────────────────────────────┤
│  Tabs (7 تبويبات مع أيقونات):                    │
│  [تكاليف] [مقارنة] [BOQ] [سوق]                  │
│  [حفر] [طرق] [شبكات]                            │
└─────────────────────────────────────────────────┘
```

- بطاقات KPI أعلى الصفحة تعرض ملخص بيانات التحليل
- اقتراحات ذكية بناءً على حالة البنود
- التبويبات تتحول لـ scrollable على الموبايل

### 3. تطبيق ألوان العلامة التجارية

الألوان المستخدمة (موجودة فعلاً في CSS variables):
- **Primary (#F3570C)**: `--primary: 16 93% 50%` ✅
- **Foreground (#161616)**: `--foreground: 0 0% 9%` ✅  
- **Secondary (#605F5F)**: `--secondary: 0 0% 37%` ✅
- **Border (#A0A09F)**: `--border: 0 0% 63%` ✅
- **Background (#FBFAFA)**: `--background: 0 17% 98%` ✅

الألوان مطبقة بالفعل. سيتم التأكد من استخدام `text-primary`, `bg-primary/10`, `border-primary/20` في البطاقات والعناصر الجديدة بدلاً من ألوان ثابتة.
