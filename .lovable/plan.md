

# إكمال الاقتراحات الذكية لشاشة Cost-Benefit Analysis وربطها مع المشاريع

## الوضع الحالي
الشاشة تحتوي على 3 اقتراحات فقط:
1. "ابدأ بإضافة أول تحليل" (عند عدم وجود تحليلات)
2. "أضف تحليل ثاني للمقارنة" (عند وجود تحليل واحد)
3. "X تحليلات غير مجدية" (عند وجود NPV سالب)

## الاقتراحات المطلوب إضافتها

| الشرط | الاقتراح | الإجراء |
|---|---|---|
| analyses.length ≥ 2 | "استخدم تحليل الحساسية لاختبار السيناريوهات" | الانتقال لـ tab sensitivity |
| analyses.length > 0 && !selectedAnalysis | "اختر تحليلاً لعرض التفاصيل الكاملة" | — |
| analysis مع assumptions فارغة | "أضف افتراضات التحليل لتوثيق أفضل" | فتح dialog التعديل |
| تحليل مع جدوى > 80% | "تحليل ممتاز! صدّره كتقرير PDF" | — (تصدير) |
| لا يوجد projectId مربوط | "اربط التحليل بمشروع للمتابعة الشاملة" | فتح dialog اختيار مشروع |
| تحليلات بـ BCR < 1 && IRR > discount | "مراجعة: BCR منخفض رغم IRR جيد" | عرض التفاصيل |

## ميزات جديدة

### 1. ربط التحليل بمشروع محفوظ
- إضافة Select في Dialog الإدخال لاختيار مشروع
- عرض اسم المشروع المرتبط في قائمة التحليلات
- فلترة التحليلات حسب المشروع

### 2. تصدير PDF للتحليل
- زر تصدير في تفاصيل التحليل
- تضمين: المؤشرات + الرسوم البيانية + الافتراضات

### 3. استيراد بيانات من المشروع
- استيراد قيمة المشروع كـ initial_investment
- استيراد عدد السنوات من مدة العقد
