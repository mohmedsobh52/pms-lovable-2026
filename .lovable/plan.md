

# المرحلة 6: تحسينات شاملة - الأمان والبحث والتجربة العامة

---

## 6.1 تحسين البحث الشامل (Global Search Enhancement)

**الملف:** `src/contexts/GlobalSearchContext.tsx` + `src/components/GlobalSearch.tsx`

حالياً البحث يعمل على الصفحات والأوامر فقط. التحسين:
- إضافة بحث مباشر في بنود المشاريع (`project_items`) من قاعدة البيانات
- إضافة بحث في العقود (`contracts`) بالاسم والرقم
- عرض النتائج في مجموعات منفصلة (صفحات / بنود / عقود)
- إضافة اختصار `Ctrl+K` للفتح السريع (إن لم يكن موجوداً)

## 6.3 تصدير PDF احترافي لتفاصيل المشروع

**ملف جديد:** `src/components/ProjectDetailsPDFExport.tsx`

مكون يولد تقرير PDF شامل لمشروع واحد يحتوي على:
- ترويسة الشركة (من letterhead-utils)
- ملخص المشروع (الاسم، التاريخ، الحالة، القيمة)
- جدول BOQ كامل مع الأعمدة العربية والإنجليزية
- رسم بياني دائري لنسب التسعير
- ملخص الفئات وقيمها
- تذييل بتاريخ الطباعة ورقم الصفحة

**الملف المرتبط:** إضافة زر "تصدير PDF" في `ProjectDetailsPage.tsx`

**الملف:** `src/components/ErrorBoundary.tsx` + صفحات متعددة

## 6.7 اختصارات لوحة المفاتيح (Keyboard Shortcuts)

**ملف جديد:** `src/hooks/useKeyboardShortcuts.tsx`

إضافة اختصارات عامة:
- `Ctrl+K`: فتح البحث الشامل
- `Ctrl+N`: مشروع جديد
- `Ctrl+S`: حفظ المشروع الحالي
- `Ctrl+E`: تصدير Excel
- `Ctrl+P`: تصدير PDF
- `Esc`: إغلاق أي dialog مفتوح
- `?`: عرض قائمة الاختصارات

**الملف المرتبط:** تفعيل الاختصارات في `App.tsx`
