
# خطة تحسين تنسيق العقد القانوني

## المشكلة الحالية
كما يظهر في الصورة، محتوى العقد (خاصة نطاق العمل) يعرض كفقرة نصية واحدة متصلة بدون تنسيق، مما يجعل القراءة صعبة وغير احترافية.

---

## التحسينات المطلوبة

### 1. توسيع الشاشة لعرض كامل
- تغيير `max-w-4xl` إلى `max-w-6xl` أو `max-w-[95vw]`
- زيادة ارتفاع المعاينة من `70vh` إلى `85vh`

### 2. تحسين عرض نطاق العمل (Scope of Work)
تحويل النص العادي إلى فقرات وبنود منسقة:
- تقسيم النص تلقائياً عند وجود `*` أو `**` أو `-` أو أرقام
- إنشاء قائمة مرقمة للأنشطة والأعمال الرئيسية
- إضافة مسافات بين الفقرات
- تمييز العناوين الفرعية بخط عريض

### 3. إضافة دالة تنسيق المحتوى
```typescript
const formatScopeOfWork = (text: string) => {
  // تقسيم النص إلى فقرات وبنود
  // التعرف على العناوين والقوائم
  // إعادة التنسيق بشكل احترافي
}
```

### 4. تحسين التنسيق العام
- زيادة المسافات بين البنود
- تحسين حجم الخط للقراءة (14px → 15px)
- إضافة ترقيم للفقرات الفرعية
- تحسين تباعد الأسطر (line-height: 2)

## خطوات التنفيذ

1. **توسيع الشاشة**: تغيير `max-w-4xl` إلى `max-w-[95vw]` و `max-h-[95vh]`
2. **إنشاء دالة التنسيق**: `formatScopeOfWork()` لتحليل وتنسيق النص
3. **تحسين عرض المحتوى**: استخدام الدالة في عرض نطاق العمل
4. **تحسين الأنماط**: إضافة CSS للفقرات والبنود
5. **تحسين الطباعة**: تحديث أنماط الطباعة
