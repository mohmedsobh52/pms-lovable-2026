

# إصلاح الرسوم البيانية وتحسين الشاشة الرئيسية

## 1. مشكلة عدم ظهور الجرافيك

**التحليل**: الرسوم البيانية تعمل عند وجود بيانات (مثل Monthly Activity)، لكن عند عدم وجود بيانات (Quotation Status / Risk Distribution) تظهر رسائل فارغة بدون أي تصور بصري.

**الحل**:
- استبدال الحالات الفارغة برسوم بيانية تجريبية (placeholder) بألوان باهتة مع نص "لا توجد بيانات - أضف بيانات للبدء"
- إضافة أيقونات متحركة في الحالات الفارغة بدلاً من النص العادي
- إضافة CTA button في كل حالة فارغة (مثلاً "أضف عرض سعر" في Quotation Status)

## 2. الاقتراحات الناقصة

**حالياً** في `DashboardPage`: 3 اقتراحات فقط (upload BOQ / open saved / view reports)
**حالياً** في `DashboardSuggestions` (داخل MainDashboard): 5 اقتراحات

**المطلوب إضافته**:

| الشرط | الاقتراح |
|---|---|
| `expiringContracts.length > 0` | "عقود تنتهي قريباً - راجعها" → `/contracts` |
| `overduePayments.length > 0` | "دفعات متأخرة تحتاج متابعة" → `/contracts` |
| `riskCount > 3` | "مخاطر متعددة نشطة - راجع خطة التخفيف" → `/risk` |
| `totalProjects > 0 && activeContracts === 0` | "اربط مشاريعك بعقود للتتبع" → `/contracts` |
| `pricingEfficiency < 50` | "أكثر من نصف البنود بدون تسعير" → `/library` |
| `totalProjects > 3 && riskCount === 0` | "فعّل تحليل المخاطر التلقائي" → `/risk` |

وإزالة التكرار بين `DashboardPage` و `DashboardSuggestions` بدمجهما.

## 3. تحسينات الشاشة الرئيسية

- **Welcome banner**: رسالة ترحيب ذكية تتغير حسب الوقت (صباح الخير/مساء الخير) مع ملخص سريع
- **تحسين الحالات الفارغة**: بطاقات تفاعلية بدلاً من النص الفارغ مع أيقونات وأزرار إجراء
- **إضافة شريط تقدم عام**: نسبة اكتمال الملف الشخصي للمشروع (هل أضاف عقود؟ مخاطر؟ عروض أسعار؟)
- **تحسين Quick Actions**: إضافة أيقونة "تحليل المخططات" و"المستخلصات"
