
# خطة: إضافة زر حذف الكل + عرض شيتات Excel كتابات منفصلة

## الميزات المطلوبة

### 1. زر حذف جميع البيانات
- إضافة زر "حذف الكل" بجانب أزرار "إضافة صف" و"استيراد ملف" في شريط أدوات الجدول
- يعرض تأكيد قبل الحذف (Dialog تأكيد)
- يحذف جميع البنود ويعيد الجدول لحالته الفارغة
- تصميم: زر أحمر مع أيقونة Trash2

### 2. تحميل ملف Excel متعدد الشيتات - كل شيت كتاب منفصل
بدلاً من دمج جميع الشيتات في جدول واحد، سيتم عرض كل شيت كتاب (Tab) منفصل:

- عند رفع ملف Excel متعدد الشيتات، يتم قراءة جميع الشيتات
- كل شيت يظهر كتاب منفصل في واجهة Tabs أعلى الجدول
- التاب الأول "البنود الأساسية" يعرض البنود الحالية
- باقي التابات تعرض بيانات كل شيت على حدة
- يمكن التنقل بين الشيتات بالنقر على التاب
- كل تاب يعرض جدول مستقل بنفس التنسيق
- زر "دمج الكل" لدمج جميع الشيتات في التاب الرئيسي

## التعديلات التقنية

### الملف: `src/pages/CostAnalysisPage.tsx`

**1. State جديد:**
```text
- sheetTabs: مصفوفة تحتوي على { name, items } لكل شيت مستورد
- activeSheetTab: اسم التاب النشط حالياً
- confirmDeleteAll: حالة فتح/إغلاق Dialog تأكيد الحذف
```

**2. زر حذف الكل:**
- يُضاف في شريط أدوات الجدول (بجانب "إضافة صف" و"استيراد ملف")
- عند الضغط يفتح Dialog تأكيد بعنوان "هل أنت متأكد من حذف جميع البنود؟"
- عند التأكيد: يُفرغ مصفوفة items ويُحذف الشيتات المستوردة

**3. تعديل handleFileImport:**
- عند رفع ملف متعدد الشيتات: بدلاً من فتح Dialog اختيار، يتم تلقائياً قراءة جميع الشيتات
- كل شيت يُحوّل لمصفوفة CostItem[] ويُحفظ في sheetTabs
- يظهر شريط Tabs أعلى الجدول مع تاب لكل شيت

**4. واجهة Tabs:**
- تظهر فقط عندما يوجد شيتات مستوردة (sheetTabs.length > 0)
- التاب الأول: "البنود الأساسية" (items الحالية)
- باقي التابات: اسم كل شيت من الملف
- زر "دمج في البنود الأساسية" في كل تاب لنقل بنوده للجدول الرئيسي
- زر "حذف التاب" لإزالة شيت مستورد

**5. imports إضافية:**
- `Tabs, TabsContent, TabsList, TabsTrigger` من مكونات UI
- `AlertDialog` للتأكيد قبل الحذف
