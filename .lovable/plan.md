

# إضافة مقارنة التكاليف + تصدير PDF/Excel لتحليل التكاليف

## 1. مقارنة تكاليف الذكاء الاصطناعي مع التكاليف الفعلية (رسم بياني)

إضافة قسم جديد بعد Pie Chart يعرض:
- رسم بياني عمودي (BarChart) يقارن سعر الوحدة الأصلي (من BOQ) مع سعر الوحدة المحسوب بالذكاء الاصطناعي لكل بند
- بطاقات ملخص: متوسط الفرق، عدد البنود الأعلى/الأقل، إجمالي الوفر المحتمل
- ألوان مميزة: أخضر للبنود الأقل تكلفة، أحمر للأعلى

## 2. تصدير التقرير إلى PDF و Excel

إضافة زرين تصدير بجوار زر "اعتماد التحليل":
- **PDF**: باستخدام jsPDF + autoTable - يشمل ملخص البطاقات + جدول البنود + رسم التوزيع
- **Excel**: باستخدام ExcelJS - يشمل sheet ملخص + sheet تفاصيل البنود + تنسيق احترافي مع letterhead

## 3. اختبار الشاشة

التحقق من عمل التحليل والترجمة والتخزين المؤقت بعد التعديلات.

## التفاصيل التقنية

### رسم المقارنة (BarChart)

```text
بند 1: [===== أصلي 150] [======= AI 180]  +20%
بند 2: [======= أصلي 200] [==== AI 160]   -20%
بند 3: [=== أصلي 100] [=== AI 105]        +5%
```

- يستخدم Recharts BarChart مع عمودين (originalPrice, aiPrice)
- يعرض فقط البنود التي لها سعر أصلي > 0
- بطاقات إحصائية: إجمالي أصلي vs إجمالي AI، نسبة الفرق، عدد البنود أعلى/أقل

### تصدير PDF

- صفحة A4 عمودية مع عنوان التقرير والتاريخ
- جدول ملخص (مواد، عمالة، معدات، إجمالي)
- جدول البنود التفصيلي (رقم، وصف، كمية، سعر وحدة، إجمالي، فرق%)
- ترقيم الصفحات

### تصدير Excel

- Sheet 1 "الملخص": بطاقات الإحصائيات (مواد، عمالة، معدات، غير مباشرة، أرباح، إجمالي)
- Sheet 2 "التفاصيل": جدول كامل بكل البنود مع التكاليف المباشرة وغير المباشرة
- تنسيق: عناوين ملونة، حدود، عرض أعمدة تلقائي

### الإضافات للكود

```text

أقسام UI جديدة:

  - بطاقات إحصائيات المقارنة
```
