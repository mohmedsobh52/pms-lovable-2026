
# خطة: نظام نسخ احتياطي + سلة محذوفات + إعادة إنشاء المشاريع المفقودة

## نظرة عامة

تنفيذ ثلاثة تحسينات رئيسية لمنع فقدان البيانات مستقبلاً:
1. إنشاء سلة محذوفات (soft delete) مع استعادة خلال 30 يوم
2. إعادة إنشاء المشاريع المفقودة (الدلم - الدرعية - المرافق) كمشاريع فارغة جاهزة
3. تعزيز تأكيدات الحذف والاستبدال بخطوات إضافية

---

## 1. سلة المحذوفات (Soft Delete)

### تغييرات قاعدة البيانات

اضافة عمودين لجدولي `saved_projects` و `project_data`:

| العمود | النوع | الافتراضي | الوصف |
|---|---|---|---|
| is_deleted | boolean | false | علامة الحذف الناعم |
| deleted_at | timestamp | null | تاريخ الحذف (للتنظيف التلقائي بعد 30 يوم) |

تحديث سياسات RLS لاستبعاد المحذوفات من الاستعلامات العادية.

### مكون جديد: `RecycleBin.tsx`

- يعرض المشاريع المحذوفة مع تاريخ الحذف والأيام المتبقية قبل الحذف النهائي
- زر "استعادة" لكل مشروع يعيد `is_deleted = false`
- زر "حذف نهائي" مع تأكيد مشدد (كتابة اسم المشروع)
- مؤشر بصري للأيام المتبقية (30 يوم)

### تحديث منطق الحذف

تعديل `SavedProjectsPage.tsx` و `projectService.ts`:
- استبدال `DELETE` بـ `UPDATE SET is_deleted = true, deleted_at = now()`
- اضافة فلتر `is_deleted = false` في جميع استعلامات جلب المشاريع
- اضافة tab/زر "سلة المحذوفات" في صفحة المشاريع

---

## 2. إعادة إنشاء المشاريع المفقودة

انشاء 3 مشاريع فارغة في قاعدة البيانات للمستخدم الحالي:

| المشروع | الوصف |
|---|---|
| الدلم | مشروع فارغ جاهز لرفع BOQ |
| الدرعية | مشروع فارغ جاهز لرفع BOQ |
| المرافق | مشروع فارغ جاهز لرفع BOQ |

سيتم الادراج في كل من `project_data` و `saved_projects` بنفس المعرف (UUID) لضمان التكامل.

## 3. تعزيز تأكيدات الحذف والاستبدال

### تحديث `SavedProjectsPage.tsx` - حوار الحذف

- اضافة خطوة كتابة اسم المشروع للتأكيد قبل الحذف
- اضافة شرح واضح: "سيتم نقل المشروع إلى سلة المحذوفات لمدة 30 يوم"
- تغيير لون زر الحذف وايقونته

### تحديث `SaveProjectButton.tsx` - حوار الاستبدال

### منطق سلة المحذوفات

```text
// الحذف الناعم - بدلاً من DELETE
UPDATE saved_projects SET is_deleted = true, deleted_at = now() WHERE id = ?
UPDATE project_data SET is_deleted = true, deleted_at = now() WHERE id = ?

// الاستعادة
UPDATE saved_projects SET is_deleted = false, deleted_at = null WHERE id = ?
UPDATE project_data SET is_deleted = false, deleted_at = null WHERE id = ?

// جلب المشاريع النشطة فقط
SELECT * FROM saved_projects WHERE user_id = ? AND (is_deleted = false OR is_deleted IS NULL)

// جلب المحذوفات (سلة المحذوفات)
SELECT * FROM saved_projects WHERE user_id = ? AND is_deleted = true AND deleted_at > now() - interval '30 days'
```
