

# إضافة اقتراحات ذكية لتحسين الأداء ودقة التحليل

## الفكرة
إضافة بطاقة اقتراحات ديناميكية في تبويب الإعداد (config) وتبويب التحليل (analysis) تقدم توصيات ذكية بناءً على الإعدادات الحالية وحالة التحليل لتحسين الأداء والدقة.

## الاقتراحات الذكية المقترحة

### اقتراحات الأداء (Performance)
| الحالة | الاقتراح | الإجراء |
|--------|----------|---------|
| chunkSize > 25 مع density عالية | قلّل حجم الدُفعة لتحسين الدقة مع الصفحات الغنية | تعيين chunkSize تلقائياً |
| quality = "high" مع صفحات > 50 | استخدم الوضع السريع أو المدمج للملفات الكبيرة | تبديل quality |
| quality = "fast" مع صفحات مخططات بصرية | الوضع النصي قد يفقد تفاصيل بصرية مهمة — جرّب المدمج | تبديل لـ hybrid |
| عدد الصفحات المختارة = 0 | اختر صفحات للتحليل من مدير PDF | فتح تبويب PDF |
| صفحات فارغة كثيرة (density=0) | استبعد الصفحات الفارغة لتوفير الوقت والتكلفة | تصفية تلقائية |

### اقتراحات الدقة (Accuracy)
| الحالة | الاقتراح | الإجراء |
|--------|----------|---------|
| depth = "fast" | التحليل السريع بدقة 70% — ارفع العمق لنتائج أدق | تبديل depth |
| OCR غير مفعّل مع مخططات بصرية | فعّل OCR لاستخراج الأبعاد والنصوص من المخططات | تفعيل OCR |
| لا توجد وحدات تحليلية مختارة | فعّل الوحدات المناسبة (هندسية/تجارية/مخاطر) | فتح الوحدات |
| أقطار أنابيب مكتشفة ولا يوجد وحدة مرافق | فعّل وحدة "مراجعة التعارضات" لشبكات الأنابيب | تفعيل الوحدة |
| نتائج بها أخطاء (error chunks) | أعد تحليل الدُفعات الفاشلة | إعادة التحليل |
| نتائج بدون دمج (> 1 دُفعة بدون merge) | ادمج النتائج للحصول على BOQ موحد | تشغيل الدمج |
