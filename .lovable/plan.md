

# خطة إضافة تصدير PDF للمستخلصات واختبار ربط المقاولين

## الوضع الحالي

### ربط المقاولين (مُنفّذ بالفعل)
- زر إضافة المقاول في شاشة التسعير (`TenderSubcontractorsTab.tsx`) يحفظ المقاول الجديد تلقائياً في جدول `subcontractors` (سطر 126-143)
- شاشة مقاولي الباطن (`SubcontractorManagement.tsx`) تقرأ من نفس الجدول
- الربط يعمل في الاتجاهين

### شاشة المستخلصات (مُنفّذة بالفعل)
- الصفحة موجودة في `/progress-certificates` مع كامل الوظائف (إنشاء، عرض، تحديث حالة، حذف)
- جداول قاعدة البيانات موجودة (`progress_certificates` و `progress_certificate_items`)

### المطلوب الآن
**إضافة تصدير PDF احترافي** للمستخلص يشمل الترويسة الرسمية والبنود والخصومات

### 1. دالة تصدير PDF

```typescript

  const doc = new jsPDF('p', 'mm', 'a4');

  const startY = addPDFLetterheadHeader(doc);
  
  // 2. عنوان المستخلص ومعلومات أساسية
  // رقم المستخلص، اسم المقاول، الفترة، الحالة
  
  // 3. جدول البنود (autoTable)
  // الأعمدة: رقم، وصف، وحدة، كمية تعاقدية، سعر، سابق، حالي، إجمالي، المبلغ
  
  // 4. ملخص مالي
  // الأعمال الحالية، السابقة، الإجمالي
  // الخصومات: احتجاز، دفعة مقدمة، أخرى
  // صافي المستحق
  
  // 5. تذييل (من letterhead-utils.ts)
  addPDFLetterheadFooter(doc, 1, 1);
  
  doc.save(`certificate-${cert.certificate_number}.pdf`);
};
```

### 2. إضافة زر التصدير

- زر PDF في جدول المستخلصات (عمود الإجراءات) بجانب زر العرض
- زر PDF في نافذة عرض المستخلص (View Dialog)

### 3. المكتبات المستخدمة (موجودة بالفعل)

- `jspdf` - لإنشاء ملف PDF
- `jspdf-autotable` - لإنشاء الجداول
- `letterhead-utils.ts` - للترويسة والتذييل الرسمي
