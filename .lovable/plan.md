

# تحسين وتحديث شاشة Cost-Benefit Analysis

## الوضع الحالي
الشاشة حالياً عبارة عن Card واحدة بسيطة تحتوي على قائمة تحليلات + تفاصيل مختارة مع رسم بياني واحد (Line Chart). التصميم أساسي بدون KPI cards علوية أو تبويبات أو مخططات متعددة أو حساسية أو مقارنة بين السيناريوهات.

## التحسينات المقترحة

### 1. إعادة هيكلة التصميم العام
```text
┌──────────────────────────────────────────────┐
│  Header (أيقونة + عنوان + زر إضافة)           │
├──────────────────────────────────────────────┤
│  4 KPI Cards (ملخص كل التحليلات)              │
│  [عدد التحليلات] [أعلى NPV] [متوسط BCR]       │
│  [أفضل IRR]                                   │
├──────────────────────────────────────────────┤
│  SmartSuggestionsBanner                      │
├──────────────────────────────────────────────┤
│  Tabs: [التحليلات | المقارنة | تحليل الحساسية] │
│                                              │
│  Tab 1: قائمة + تفاصيل (محسّنة)               │
│  Tab 2: مقارنة جنب لجنب بين التحليلات         │
│  Tab 3: تحليل حساسية (ماذا لو)               │
└──────────────────────────────────────────────┘
```

### 2. بطاقات KPI علوية
| البطاقة | القيمة |
|---|---|
| عدد التحليلات | `analyses.length` |
| أعلى NPV | أفضل تحليل من حيث صافي القيمة الحالية |
| متوسط BCR | متوسط نسبة الفائدة للتكلفة |
| أفضل IRR | أعلى معدل عائد داخلي |

### 3. تبويب المقارنة (جديد)
- جدول مقارنة بين كل التحليلات المحفوظة (NPV/BCR/IRR/Payback)
- Radar Chart لمقارنة الأداء المالي
- Bar Chart مجمع للاستثمار vs الفوائد vs التكاليف

### 4. تبويب تحليل الحساسية (جديد)
- اختيار تحليل موجود
- تغيير معدل الخصم (±5%) ومشاهدة تأثيره على NPV فوراً
- تغيير الفوائد السنوية (±20%) ومشاهدة التأثير
- رسم بياني يوضح NPV عبر نطاق من discount rates (Tornado/Waterfall chart)

### 5. تحسين تبويب التفاصيل
- إضافة BarChart للفوائد vs التكاليف بجانب LineChart الحالي
- إضافة مؤشر بصري (Gauge/Progress) لقوة الجدوى
- عرض الافتراضات والمخاطر والتوصيات في Accordion
- إضافة زر تصدير PDF

### 6. تحسين Dialog الإدخال
- تقسيم الحقول لخطوات (Stepper): بيانات أساسية → مالية → ملاحظات
- إضافة tooltips توضيحية لكل حقل

### 7. ألوان العلامة التجارية
استخدام `text-primary`, `bg-primary/10`, `border-primary/20` بدلاً من purple/pink الحالية.
