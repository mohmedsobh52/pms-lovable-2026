

# تفعيل وتحسين حفظ المشروع مع منع تكرار الأسماء

## المشكلة الحالية

1. زر "حفظ المشروع" في تبويب Analyze BOQ يحفظ فقط في جدول `saved_projects` ولا يحفظ البنود في `project_data` و `project_items`، مما يعني أن المشروع لا يظهر بشكل كامل في قائمة المشاريع
2. يمكن حفظ مشاريع بنفس الاسم دون تحذير واضح
3. لا يتم الانتقال تلقائياً لتبويب المشاريع بعد الحفظ

## الحل المقترح

### الملف: `src/components/SaveProjectDialog.tsx`

**التغييرات:**

1. **حفظ شامل في كلا الجدولين:**
   - عند الحفظ، إنشاء سجل في `saved_projects` + `project_data` + `project_items` (البنود المفصّلة)
   - هذا يضمن ظهور المشروع في قائمة المشاريع مع كل البنود

2. **منع تكرار الأسماء بشكل صارم:**
   - فحص الاسم في كلا الجدولين `saved_projects` و `project_data`
   - عرض رسالة خطأ واضحة عند وجود اسم مكرر مع اقتراح تغيير الاسم
   - إضافة علامة تحذير حمراء مباشرة تحت حقل الاسم عند كتابة اسم موجود (فحص فوري أثناء الكتابة)

3. **تحسين واجهة الحوار:**
   - إضافة فحص فوري (debounced) أثناء كتابة الاسم لعرض تحذير "الاسم موجود مسبقاً"
   - تعطيل زر الحفظ عند وجود اسم مكرر
   - إضافة دعم ثنائي اللغة (عربي/إنجليزي) للرسائل

4. **بعد الحفظ الناجح:**
   - استدعاء `onSaved` مع معرف المشروع
   - هذا يُفعّل الانتقال لتبويب المشاريع في `SavedProjectsPage`

### الملف: `src/components/BOQAnalyzerPanel.tsx`

**التغيير:** التأكد من أن `handleProjectSaved` يعمل بشكل صحيح ويعيد تحميل المشاريع + ينتقل لتبويب المشاريع.

## التفاصيل التقنية

### منطق الحفظ المحسّن

```text
1. فحص اسم المشروع (فوري أثناء الكتابة):
   - استعلام saved_projects + project_data بالاسم
   - عرض تحذير فوري إذا كان الاسم مستخدماً

2. عند النقر على "حفظ":
   - إنشاء سجل في project_data (مع analysis_data, wbs_data, total_value, items_count)
   - إنشاء سجل في saved_projects (بنفس الـ id)
   - إنشاء سجلات في project_items (البنود المفصّلة مع sort_order)
   - استدعاء onSaved(projectId)

3. عند وجود اسم مكرر:
   - تعطيل زر الحفظ
   - عرض رسالة "هذا الاسم مستخدم بالفعل، اختر اسماً آخر"
```

### فحص الاسم الفوري (Debounced Check)

```text
// عند تغيير اسم المشروع
useEffect with 500ms debounce:
  - إذا كان الاسم فارغاً: لا فحص
  - استعلام saved_projects + project_data
  - تحديث حالة nameExists: boolean
  - عرض/إخفاء رسالة التحذير
```
