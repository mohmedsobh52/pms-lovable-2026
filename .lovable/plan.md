

# تحسين شامل للمستخلصات: تقرير PDF + مقارنة متتالية + إشعارات تلقائية

## 1. التحقق من ربط المستخلصات بالعقود (مكتمل بالفعل)

ربط المستخلصات بالعقود يعمل بشكل صحيح:
- `NewCertificatePage.tsx`: عند اختيار عقد يتم جلب `contractDetails` و `contractPayments` وعرضهما في بطاقات مخصصة (سطور 250-268, 484-576, 612-657)
- `ProgressCertificatesPage.tsx`: حوار العرض يجلب بيانات العقد والدفعات (سطور 214-236, 637-700)
- **لا يحتاج تعديل**

## 2. تقرير PDF شامل للمستخلص مع تفاصيل العقد

### الملف: `src/pages/ProgressCertificatesPage.tsx`

تحسين دالة `handleExportPDF` الحالية (سطر 259-301) لتشمل:

**صفحة 1 - الغلاف والبيانات:**
- عنوان التقرير مع رقم المستخلص
- بيانات المقاول والمشروع والفترة
- **بطاقة تفاصيل العقد** (إذا مرتبط): رقم العقد، القيمة، نوعه، نسبة الاحتجاز، شروط الدفع

**صفحة 2 - الشروط التعاقدية:**
- `terms_conditions` و `payment_terms` من العقد المرتبط
- `scope_of_work` إن وجد

**صفحة 3 - الدفعات السابقة:**
- جدول الدفعات (`contract_payments`) مع الحالة والمبالغ
- ملخص: مدفوع / معلق / متأخر

**صفحة 4+ - جدول البنود:**
- الجدول الحالي كما هو (رقم، وصف، وحدة، كمية، سعر...)

**الصفحة الأخيرة - الملخص المالي:**
- الأعمال الحالية والسابقة والإجمالية
- الخصومات (احتجاز + مقدمة + أخرى)
- صافي المستحق
- **نسبة الإنجاز من العقد** (total_work_done / contract_value)

### التعديلات:
- تعديل `handleExportPDF` لقبول بيانات العقد والدفعات كمعاملات إضافية
- تعديل `handleExportCertificatePDF` لجلب بيانات العقد والدفعات قبل التصدير
- إضافة صفحات العقد والشروط والدفعات في PDF

## 3. لوحة مقارنة المستخلصات المتتالية

### الملف: `src/pages/ProgressCertificatesPage.tsx`

إضافة حوار مقارنة جديد يظهر عند اختيار مستخلصين لنفس المقاول:

**زر التفعيل:**
- إضافة زر "مقارنة المستخلصات" في شريط الإجراءات العلوي
- يفتح حوار لاختيار مستخلصين (الحالي والسابق) - يتم تعبئة آخر مستخلصين تلقائيا

**محتوى المقارنة:**
- **ملخص الفروقات**: إجمالي الفرق في المبالغ، عدد البنود المتغيرة
- **جدول تفصيلي**: لكل بند يعرض:
  - الكمية في المستخلص 1 vs المستخلص 2
  - الفرق في الكمية (+ / -)
  - الفرق في المبلغ
  - نسبة التغيير %
- **ترميز لوني**: أخضر للزيادة، أحمر للنقصان، رمادي بدون تغيير

### التعديلات:
- إضافة state: `showCompareDialog`, `compareData`
- إضافة دالة `handleCompareCertificates` لجلب بنود المستخلصين ومقارنتها
- إضافة مكون Dialog للمقارنة مع الجدول والملخص

## 4. إشعارات تلقائية

### الملف: `src/pages/ProgressCertificatesPage.tsx`

**أ) إشعار اقتراب موعد استحقاق دفعة:**
- عند تحميل الصفحة، فحص `contract_payments` ذات الحالة `pending`
- إذا كانت `due_date` خلال 7 أيام أو أقل، إظهار إشعار تحذيري
- استخدام `useNotifications` hook الموجود

**ب) إشعار تجاوز الكميات:**
### الملف: `src/pages/NewCertificatePage.tsx`
- عند تعديل كمية بند (`updateItemQuantity`)، إذا تجاوز `total_quantity` الكمية المتعاقد عليها `contract_quantity`
- إظهار إشعار تحذيري فوري مع اسم البند ونسبة التجاوز
- إضافة ملخص عدد البنود المتجاوزة أعلى الجدول

### التعديلات:
- استيراد `useNotifications` في كلا الملفين
- إضافة `useEffect` في `ProgressCertificatesPage` لفحص الدفعات القادمة عند التحميل
- تعديل `updateItemQuantity` في `NewCertificatePage` لإضافة تنبيه التجاوز
- إضافة شريط تحذير أعلى جدول البنود يعرض عدد البنود المتجاوزة
