## التفاصيل التقنية

### 1. تحديد المشروع في ContractsPrintPreview

إضافة في قسم الإعدادات:
```typescript
// Project filter
const [selectedProjectId, setSelectedProjectId] = useState("all");

// جلب المشاريع
const { data: projects } = useQuery({
  queryKey: ['projects-for-filter'],
  queryFn: async () => {
    const { data } = await supabase
      .from('project_data')
      .select('id, name')
      .eq('user_id', user?.id);
    return data || [];
  }
});

// تصفية العقود حسب المشروع
const projectFilteredContracts = selectedProjectId === "all"
  ? filteredContracts 
  : filteredContracts.filter(c => c.project_id === selectedProjectId);
```

### 2. مكون LegalContractPreview الجديد

```typescript
interface LegalContractPreviewProps {
  open: boolean;
  onOpenChange: (open: boolean) => void;
  contract: Contract;
}

// أقسام العقد القانوني:
const sections = {
  header: true,           // ترويسة الشركة
  parties: true,          // الأطراف
  subject: true,          // موضوع العقد (نطاق العمل)
  value: true,            // قيمة العقد
  duration: true,         // مدة العقد
  financialTerms: true,   // الشروط المالية
  paymentTerms: true,     // شروط الدفع
  additionalNotes: true,  // ملاحظات إضافية
  signatures: true,       // التوقيعات
  footer: true            // التذييل مع QR
};
```

### 3. تصدير PDF بتنسيق قانوني

استخدام `jsPDF` مع تنسيقات خاصة:
- خطوط عربية احترافية (Cairo / Amiri)
- ترقيم البنود (الأول، الثاني، الثالث...)
- إطارات وخطوط فاصلة
- مساحات للتوقيعات
- QR Code يحتوي رابط العقد
