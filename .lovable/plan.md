

# إصلاح مشكلة عدم ظهور المشاريع المحفوظة

## المشكلة الجذرية

### التخزين المؤقت (Session Cache) يمنع ظهور المشاريع الجديدة

في `SavedProjectsPage.tsx` (سطر 107-122)، عند فتح صفحة المشاريع:
1. يتحقق أولاً من `sessionStorage` — إذا وجد بيانات مخزنة مؤقتاً (أقل من 3 دقائق) يعرضها **ويتوقف بدون الاتصال بقاعدة البيانات**
2. عندما يحفظ المستخدم مشروعاً من صفحة أخرى (تحليل BOQ أو تحليل المخططات)، لا يتم مسح الكاش
3. النتيجة: عند العودة لصفحة المشاريع، يظهر الكاش القديم بدون المشروع الجديد

```text
المسار الحالي:
  حفظ مشروع (صفحة أخرى) → الانتقال للمشاريع → كاش قديم موجود → عرض بيانات قديمة → return ❌

المطلوب:
  حفظ مشروع → الانتقال للمشاريع → عرض كاش + جلب بيانات جديدة بالخلفية → تحديث ✅
```

## الإصلاح

### الملف: `src/pages/SavedProjectsPage.tsx`

**1. تغيير استراتيجية الكاش (Stale-While-Revalidate):**
- عرض بيانات الكاش فوراً للسرعة
- جلب البيانات الجديدة من قاعدة البيانات في الخلفية دائماً
- تحديث الشاشة عند وصول البيانات الجديدة
- إزالة `return` المبكر بعد قراءة الكاش

**2. مسح الكاش عند الوصول من حفظ مشروع:**
- إذا كان `newProjectState` موجوداً، تجاوز الكاش مباشرة
