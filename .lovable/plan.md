

# تحديث شامل لشاشة تحليل المخططات — دقة + أداء + ملفات مجمعة

## 1. تحسين دقة حسابات المواسير وأنواعها

### الملف: `src/pages/DrawingAnalysisPage.tsx`

**تحسين `SAR_REF_2025`:**
- إضافة أنواع مواسير جديدة: UPVC، PP (بولي بروبيلين)، Corrugated HDPE، Steel pipes
- إضافة أقطار إضافية مفقودة (Ø75, Ø125, Ø355, Ø450, Ø560, Ø630, Ø710, Ø800 لـ HDPE)
- إضافة وصلات المواسير (fittings): كوع، تي، ريديوسر، فلانجة، وصلات جيبلاند
- إضافة أعمال حماية المواسير: خرسانة إحاطة، لبادة، شريط تحذيري

**تحسين `extractPipeDiameters`:**
- دعم أنماط إضافية: `OD`, `NB`, `NPS`, `قطر خارجي`, `قطر داخلي`
- استخراج نوع المادة مع القطر (PVC/HDPE/DI/GRP/RCP)
- إرجاع كائن `{diameter, material, type}` بدلاً من رقم فقط

**إضافة `extractPipeDetails` جديدة:**
- استخراج: قطر + مادة + PN/SN class + طول + عمق تركيب
- دعم جداول الأنابيب المنسقة في PDF

### تحسين System Prompts:
- تعديل `SYS_VISUAL_INFRA` لطلب تفصيل مادة الأنبوب ومواصفاته (PN/SN/SDR)
- إضافة تعليمات صريحة لحساب أطوال المواسير من القطاعات الطولية
- طلب حساب الوصلات والملحقات لكل خط أنابيب

## 3. إضافة رفع ملفات مجمعة مع تصنيف تلقائي

### الملف: `src/pages/DrawingAnalysisPage.tsx`

**إضافة دعم Multi-file + Folder upload في تبويب PDF:**
- تعديل `handleFiles` لدعم `webkitdirectory` (مجلدات)
- إضافة قائمة ملفات مرفوعة مع حالة كل ملف
- تصنيف تلقائي لكل ملف بناءً على الاسم والمحتوى:
  - أسماء تحتوي `plan/مسقط` → PLAN
  - أسماء تحتوي `profile/طولي` → PROFILE
  - أسماء تحتوي `section/عرضي` → SECTION
  - أسماء تحتوي `detail/تفصيل` → DETAIL
  - أسماء تحتوي `boq/كمي` → BOQ

**إضافة مكون `BatchFileManager` (inline):**
- عرض جميع الملفات المرفوعة مع التصنيف التلقائي
- إمكانية تعديل التصنيف يدوياً لكل ملف
- زر "تحليل الكل" لتحليل جميع الملفات بالتتابع
- شريط تقدم إجمالي + حالة كل ملف (قيد الانتظار/جاري/مكتمل/خطأ)
- دمج نتائج جميع الملفات في تقرير موحد

**تعديل input file:**
- إضافة زر "📁 رفع مجلد" بجانب "📄 رفع ملف PDF"
- `<input webkitdirectory directory multiple>` لدعم المجلدات
