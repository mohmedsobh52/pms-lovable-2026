
# خطة: إضافة قسم الاقتراحات الذكية للوحة الإدارة

## الفكرة
إضافة قسم "اقتراحات وتوصيات" ذكي يظهر في لوحة الإدارة، يعرض توصيات مبنية على حالة البيانات الفعلية (مثلاً: إذا لا يوجد مشاريع، يقترح إنشاء مشروع جديد). يتغير المحتوى ديناميكياً حسب الإحصائيات.

## التعديلات

### الملف: `src/pages/AdminDashboardPage.tsx`

اضافة قسم "اقتراحات وتوصيات" بعد بطاقات الإحصائيات وقبل الإجراءات السريعة، يتضمن:

**اقتراحات ديناميكية حسب الحالة:**

| الشرط | الاقتراح (عربي) | الاقتراح (English) | الإجراء |
|---|---|---|---|
| المستخدمين = 0 | ادعُ مستخدمين جدد للنظام | Invite new users | انتقل لـ /admin/users |
| المشاريع = 0 | أنشئ أول مشروع لتحليل بنود الأعمال | Create your first BOQ project | انتقل لـ /new-project |
| العقود = 0 | أضف عقوداً لتتبع الالتزامات المالية | Add contracts to track commitments | انتقل لـ /contracts |
| عروض الأسعار = 0 | ارفع عروض أسعار للمقارنة والتحليل | Upload quotations for comparison | انتقل لـ /quotations |
| القوالب = 0 | أنشئ قوالب BOQ لتسريع العمل | Create BOQ templates | انتقل لـ /templates |
| نشط(30d) = 0 ومشاريع > 0 | راجع المشاريع القائمة وقم بتحديثها | Review and update existing projects | انتقل لـ /saved-projects |

**تصميم القسم:**
- بطاقة بعنوان "اقتراحات وتوصيات" مع أيقونة Lightbulb
- كل اقتراح يظهر كصف مع أيقونة ملونة + نص + زر إجراء
- يختفي القسم بالكامل إذا لم تكن هناك اقتراحات (كل الإحصائيات > 0)
- لون خلفية مميز (أصفر/ذهبي خفيف) لجذب الانتباه

### التفاصيل التقنية
- حساب الاقتراحات عبر `useMemo` يعتمد على كائن `stats`
- كل اقتراح يحتوي: `{ icon, text, textAr, action: () => navigate(path), color }`
- لا حاجة لتعديل قاعدة البيانات أو Edge Functions
- ملف واحد فقط يتم تعديله: `AdminDashboardPage.tsx`
